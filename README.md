# Truth Lens – AI Generated Media Detector

## 🔍 Problem Statement
Detect AI-generated images, videos, and text on social media platforms in real-time.

## 💡 Solution
Truth Lens uses AI models to analyze uploaded media and classify it as:
- Real
- AI Generated
- Suspicious

It provides a confidence score and helps prevent fake content, scams, and misinformation.

## ⚙️ Features
- Image upload and detection
- Confidence score output
- Real-time API processing
- Scalable backend architecture

## 🧠 Tech Stack
Frontend: React.js  
Backend: Flask (Python)  
AI: CNN / Deep Learning Models  

## 🏗️ Architecture
User Upload → API → AI Model → Result (Confidence Score)
